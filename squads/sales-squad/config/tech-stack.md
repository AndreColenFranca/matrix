# Tech Stack - Sales Squad

## Core

- **Language:** TypeScript / JavaScript
- **Runtime:** Node.js 20+

## E-commerce

- **Payment Gateway:** Stripe / PayPal
- **Shopping Cart:** Custom or Shopify
- **Product Database:** PostgreSQL / MongoDB
- **Cache:** Redis

## Services

- **Email:** SendGrid / AWS SES
- **Shipping:** EasyPost / ShipStation
- **Analytics:** Segment / Mixpanel

## Development

- **Testing:** Jest / Vitest
- **Linting:** ESLint
- **Code Formatting:** Prettier
- **CI/CD:** GitHub Actions

## Libraries

- Stripe SDK for payments
- Express.js for APIs
- Joi for validation
- Winston for logging
